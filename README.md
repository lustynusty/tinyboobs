# Telegram Shop Bot 🛍️

Бот для продажи цифровых и физических товаров с оплатой через звёзды Telegram.

## ✨ Возможности

- 📸 Продажа цифровых товаров (фото) - автоматическая отправка после оплаты
- 👙 Продажа физических товаров (бельё) - уведомление админа с адресом доставки
- ⭐ Оплата через звёзды Telegram
- 🛠 Админ-панель с полным управлением товарами
- 👁 Просмотр фото товаров админом бесплатно
- 📋 Добавление, удаление, изменение цен товаров

## 🤖 Команды бота

- `/start` - Запуск бота и главное меню

## 🛠 Админ-панель

Доступна только для пользователей, указанных в `ADMIN_IDS`. Позволяет:

- Добавлять цифровые и физические товары
- Просматривать список товаров
- Изменять цены
- Удалять товары
- Просматривать фото товаров (бесплатно)
- Получать уведомления о заказах физических товаров
